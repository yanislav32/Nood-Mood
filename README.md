# NØØD landing

Одностраничный сайт-заготовка для NØØD. Десктоп рассчитан на один экран без скролла, мобильная версия скроллится.

## Если у тебя на ПК ничего нет

1. Открой репозиторий на GitHub.
2. Нажми **Code → Codespaces → Create codespace**.
3. Когда откроется VS Code в браузере, внизу открой **Terminal**.
4. Введи:

```bash
npm run build
npm run dev
```

5. Codespaces предложит открыть порт `5173` — нажми **Open in Browser**.

## Если запускаешь локально

Нужен Node.js и Python 3. В папке проекта выполни:

```bash
npm run build
npm run dev
```

После запуска открой:

```text
http://localhost:5173
```

## Что редактировать

- `index.html` — структура страницы: боковое меню, hero, карточки, footer.
- `src/styles.css` — весь визуал: цвета, сетки, адаптив, неоновый фон, карточки.
- `src/main.js` — тексты разделов и логика переключения меню.

## Текущий статус

Сайт намеренно использует CSS-заглушки вместо настоящих изображений. Когда появятся реальные логотип, продуктовые фото и фоновые изображения, их нужно будет заменить в hero и карточках.
